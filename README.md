# File-Converter-Telegram-Bot

Can convert Images, Videos, Audios, Fonts, Documents and Ebooks.

see the bot in action @filesconvertrobot (https://t.me/filesconvertrobot)

# Env variables

TOKEN - Bot API Token

ID - API ID

HASH - API Hash

OWNERID - Bot Owner ID
